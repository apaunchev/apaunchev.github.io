UltraLinx Themes

In 2012, I teamed up with Oliur Rahman and we started making themes. Since then, I have worked on a number of Tumblr and WordPress themes, some of which have sold hundreds of copies and have been featured on ThemeForest and Tumblr’s Theme Garden.

pravopis.be

pravopis.be is a spell-checking app for commonly misspelled words and phrases (in Bulgarian only). Designed and built over a weekend, it was the perfect excuse to experiment with JavaScript while building a useful tool for myself (and hopefully others). The database currently holds nearly 1000 words, and I’m adding more all the time.

init

init is a Sass-based, object-oriented CSS framework. Or in other words, my personal CSS playground. Fork at your own risk.
