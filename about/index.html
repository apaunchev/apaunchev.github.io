My name is Angel Paunchev and I’m a web developer from Bulgaria. I build tools for the web and write about design, development, technology, and more.

Timeline

In 2011, I started a BSc Computing course at Glasgow Caledonian University.

In 2012, I teamed up with Oliur Rahman and we started making themes. That was also the year I started wearing glasses.

In 2013, I launched pravopis.be.

This site

I’ve had this site for over 5 years and its purpose has changed a lot over this time. At the moment, I use it as a place to showcase my recent work and to give perspective and insight on topics that interest me.

It might sound selfish, but this site exists purely for me. It has helped me grow in many ways with each of its iterations. If it can do the same for you, I’d be more than happy.

Built with Sublime Text and Jekyll. Source code is hosted on GitHub.
